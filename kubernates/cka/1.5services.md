## Create a new service to access the web application using the service-definition-1.yaml file.

```
apiVersion: v1
kind: Service
metadata:
  name: webapp-service
  namespace: default
spec:
  ports:
  - nodePort: 30080
    port: 8080
    targetPort: 8080
  selector:
    name: simple-webapp
  type: NodePort
```
`kubectl apply -f sample.yaml`
